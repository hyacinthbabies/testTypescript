## 环境
1. 安装node.js
`参考：https://www.runoob.com/nodejs/nodejs-install-setup.html`
2. 在终端进入该项目的根目录执行命令
`npm install`安装项目所需要的库
3. 在终端进入该项目的根目录执行命令
`npm run start`开启项目
4. 在浏览器输入
`http://localhost:8080`打开网页首页，
 输入`http://localhost:8080/admin`打开后台管理页面
